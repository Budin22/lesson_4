Селекторы и их комбинации

Селектор елемента(тега)

p{color: blue} - Использывует свойства ко всем заданым тегам к примеру "р"

Селектор #id

Селектор идентификатор должен быть уникальным и меняет свойства только с атрибутом id(a href="#" id="only")

Селектор .class

К тегу добавляется атрибут class, который позволяет менять свойства всех тегов с данным атрибутом (p class="all")

Селекторы по нескольким класам(.class1.class2)

Используется когда к одному тегу необходимо определить несколько классов(p class="two all")

С приоритетом к последнему позже написаному коду(ниже)

Вложеные селекторы(class class)

Создается стиль для тега только в том случае когда он находится внутри другого тега(p li)

Примеры

  • футбол
  • Теннис
  • Хокей
  • Шахматы
  • Стрельба с лука
